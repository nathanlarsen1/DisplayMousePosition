<h1>Display Mouse Position</h1>


<h2>Description</h2>
The project consists of an AutoIt script that displays the mouse position on the screen next to the mouse pointer.<br/>

<h2>Language</h2>

- <b>AutoIt</b>

<h2>Environments Used </h2>

- <b>Windows 10 and 11</b>
